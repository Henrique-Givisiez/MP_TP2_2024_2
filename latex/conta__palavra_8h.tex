\doxysubsection{Descrição detalhada}
Declaração de funções e estruturas para contar palavras em arquivos de texto. 

Este arquivo contém definições de estruturas e protótipos de funções utilizadas para processar textos, contar palavras, ordenar resultados e gerar strings formatadas.
